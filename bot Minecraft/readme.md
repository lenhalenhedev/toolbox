# Node.js bot Minecraft 

A small personal Node.js project.  
Shared for reference and reuse. No guarantees, no support.

## Requirements
- Node.js 16+ (18+ recommended)

## Files

. ├── index.js ├── package.json └── config.txt

## Setup
Edit `config.txt` with your own values.

Do **not** commit real tokens or secrets.

## Run
``npm install``and 
``node index.js``

Built for personal use
